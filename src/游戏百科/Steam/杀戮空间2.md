# 杀戮空间2 Windows服务器搭建指南

## 服务器启动命令

使用以下命令启动杀戮空间2服务器：

```cmd
start .\Binaries\win64\kfserver kf-bioticslab?GamePassword=?adminpassword=123?port=12345?PeerPort=12356
```

::: warning 重要提示
请确保：

- 替换所有密码和端口号
- 保留所有特殊符号（?、=等）
- 防火墙开放相应端口
:::
